# Nexus API - NestJS
API desarrollada con **NestJS**, **TypeORM** y **PostgreSQL** para la gestión de usuarios.

## Stack tecnológico

- NestJS
- TypeORM
- PostgreSQL
- Yarn
- class-validator
- class-transformer
- bcrypt

---

## Requisitos

- Node.js v20.20.1
- Yarn
- PostgreSQL
- Docker (opcional, para la base de datos)

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
